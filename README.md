# JogoDaForca

![](https://imgur.com/SrA1mr3.gif)


## Introdução

Este jogo da forca permite o usuario selecionar duas categorias de palavras,
que serão geradas aleatoriamente e o usuario terá algumas chances para tentar acertar a 
palavra.

## Funcionalidades

- Permite seleção da categoria de palavras (Frutas ou Animais).
- O usuario terá 6 chances para acertar a palavra.
- Não é permitido repetir letras de chutes.
- Na tela serão exibidos os chutes do usuário.
- Na tela será exibido o bonequinho em progresso para ser 'enforcado' conforme o usuario erra.
